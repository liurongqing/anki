1. 下载 `Anki App`

   https://apps.ankiweb.net/

2. 安装 Anki Connect 插件

   工具 -> 插件 -> 获取插件 -> 粘贴插件代码

   https://ankiweb.net/shared/info/2055492159

3. 安装 `Anki for VSCode`

4. 打开命令 `cmd + shift + p`

   1. `Anki: Send To Deck`

      发送到 `notes` 组中， 一级标题会生成标签

   2. `Anki: Send To Own Deck`

      发送到自定义组中， 自定义一级标题为组名

   3. `Anki: Send To DirName Deck`

      发送以当前目录名为组名
